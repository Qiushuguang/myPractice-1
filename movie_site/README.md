

## 一个简单的多页面动态点用网站
---
### 项目简介
根据慕课网《node＋mongodb 建站攻略》一期，实现了简单的多页面动态电影信息站点，
共实现了电影预览首页、电影详细信息页、电影后台录入页（包含更新功能）、电影列表页四个页面
其中列表页可以实现电影信息修改的跳转 和 电影信息删除的功能。

---
### 使用技术

- node.js平台
- express框架
- mongodb 数据库
- jade 模版引擎
- underscore 工具库
- bodyParser 中间件
- bootstrap 
- jquery

---
### 安装说明：
需要安装node,预先安装好mongodb数据库并配置好环境变量。
- 使用`bower install`安装前端静态资源，
- 使用`npm install` 安装依赖文件.
